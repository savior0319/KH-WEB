@ 키프레임 사용
-> 키 프레임 사용하면 애니메이션의 중간 설정을 줄 수 있음

## 애니메이션 반복하기 ##
-> animation-iteration-count 사용하면 애니매이션이 반복 됨

## 애니메이션 앞뒤로 움직이기 ##
-> animation-direction 속성을 alternate로 지정하면 앞뒤로 반복 됨

@ transform
-> transform을 이용하면 회전, 크기 조절, 이동, 기울기 등의 여러 작얼을 할 수 있음
-> 다양한 속성을 가지고 2D나 3D를 표현 할 수 있음

## rotate ##
-> 지정한 각도만큼 요소를 시계방향(양수값)이나 반대방향(음수값)으로 회전 시키는 함수

선택자 {
  transform : rotate(값 deg)
}
