## table 병합 속성 ##

rowspan : <td> 태그의 속성으로 지정한 행만큼 행을 병합
colspan : <td> 태그의 속성으로 지정한 열만큼 열을 병함

사용법

<td rowspan='숫자'></td>
<td colspan='숫자'></td>

## 그외 테이블 관련 태그 ##

테이블을 각각 나누어서 활용 할 수 있음

<thead>
-> 테이블 구조를 나눌때 머리 부분 태그
-> 한개만 존재 할 수 있음

<tbody>
-> 테이블 구조를 나눌때 몸체가 되는 부분
-> 여러개가 존재 할 수 있음

<tfoot>
-> 테이블 구조를 나눌때 마지막이 되는 부분
-> 한개만 존재 할 수 있음

@ 페이지 영역 태그의
-> 페이지를 특정 영역으로 분할 하여 사용할 수 있는 태그의

## <iframe> 태그 ##
-> 웹 문서안에 다른 웹페이지를 추가하는 태그

## <div> 태그 ##
-> block 형식의 공간을 분할
-> 공간 단위로

## <span> 태그 ##
-> inline 형식의 공간을 분할
-> Text 단위로

@ html4 페이지 구조
-> html4에서는 페이지 구조를 <div>, <span> 태그를 이용하여 구조를 만듦
-> 구역을 설정하고 거기에 id값을 넣어서 구분 함

※ 단점
-> 사용자마다 id값을 다르게 부여하여 구조를 이해하는데 어려움이 있어서 차후 유지 보수가 어려워 짐
-> 검색 엔진이 웹 소스를 확인할 때도 메뉴, 본문 구분이 어려워짐

ex)
<div id="header">
</div>

<div id="contents">
</div>

<div id="footer">
</div>

@ html5 페이지 구조
-> 시멘틱 구조를 활용한 구조
-> 시멘틱 태그를 사용하여 구조를 잡음
        - 페이지 구조를 특정 기능에 맞는 태그를 사용하여 구분
        - 시멘틱 태그를 쉽게 파악하고 좀 더 정확한 정보를 검색 할 수 있게 함
        - header, section, footer
-> 웹 접근성의 용이함
        - 장애인, 고령자 등이 웹 사이트에서 제공하는 정보에 비장애인과 동등하게 접근하고 이해할 수 있도록 보정하는 것
## <header> ##
-> 특정 부분의 머리말로 주로 검색에(form), 메뉴(nav)를 넣음
        - <nav>
        - 다른 사이트나 페이지로 이동하는 태그를 모아 놓은 태그
        - 특정 태그에 종속되지 않고 어느곳에서나 사용할 수 있음
        - 주로 메뉴, footer의 사이트 링크 모음에 많이 쓰임

## <section> ##
-> 웹 문서에서 컨텐츠가 들어가는 영역, 컨텐츠를 주제별로 묶을 때 사용
- <article>
- 웹 페이지의 내용이 들어가는 영역
- 이 태그 영역은 다른 곳으로 배포하거나 재사용이 가능
- 검색 로봇은 이 태그가 사용된 컨텐츠는 배포 할 수 있는 컨텐츠로 인식

## <aside> ##
-> 사이드 바 라고 부름
-> 본문 외에 기타내용을 담고 있는 영역
-> 주로 광고를 달거나 링크모음 등을 표현

## <footer> ##
-> 웹 페이지의 맨 아래쪽에 위치하며, 회사소개, 저작권, 연락처 등의 정보를 표시

## <img> 태그 ##
-> 웹 페이지에 사진이나 그림을 삽입할 때 사용하는 태그

속성
src              : 삽입할 이미지 경로를 지정하는 속성
alt              : 이미지를 설명해주는 텍스트 속성
width, height    : 이미지의 크기를 설정하는 속성

작성법

<img src='경로' alt='설명 문구' width='너비' height='길이'> 


## 이미지 맵 태그 ##
-> <map> <area>를 사용하여 이미지를 구역별로 나누어 링크를 걸어주는 태그

속성
alt : 대체 텍스트 지정
coords : 분할할 지역 좌표 지정(시작, 끝)
href : 링크 페이지 경로 지정
media : 링크 페이지 최적화 미디어 지정
rel : 현재 페이지, 링크 페이지 관계 지정
shape : 링크로 사용할 영역 형태 지정 (default, rect, circle, poly)
target : 링크 페이지가 열릴 위치 (새창, 현재창, 부모창, 최상위창)
type : 링크 페이지의 미디어 유형 지정

작성법
<img src='경로' usemap='#tt'>
<map name='tt'>
        <area shape='모양' coords='00.00(시작),00.00(끝) href='링크경로' target='위치'>
        <area shape='모양' coords='00.00(시작),00.00(끝) href='링크경로' target='위치'>
</map>

## <audio> 태그 ##
-> 웹 브라우저에서 플러그인의 도움없이 음악을 재생 할 수 있게 만들어주는 태그

속성 

src : 음악 파일의 경로 지정
controis : 재생도구 출력 지정
autoplay : 자동 재생여부 지정 (모바일 적용 X)
loop : 반복 여부 지정


## <video> 태그 ##
-> 웹 브라우저에서 플러그인의 도움 없이 미디어를 재생할 수 있게 만들어 주는 태그가

속성

src : 미디어 파일에 대한 경로 지정
controis : 재생도구 출력 여부 지정
autoplay : 자동재생 여부 (모바일 적용 X)
loop : 반복재생 여부
width : 너비를 지정
height : 높이를 지정

※ 비디오 형식은 mp4로 지정

## <a> 태그 ##
-> 페이지에서 해당 부분을 클릭하면 지정된 페이지로 이동하는 태그
-> 외부 사이트 연결, 문서 내부에서 이동 가능함

속성
href : 링크한 페이지의 id값이나 사이트 주소 지정
target : 링크 페이지가 표시될 위치(새창, 현재창) 저장 가능
        새창 : _blank
        현재창 : _self


@ form 태그
-> 사용자가 입력한 data를 보내는 방식과 처리할 프로그램을 정하는 태그

속성 
action : 데이터를 처리할 페이지를 지정
method : 데이터를 전송할 방식을 지정(get과 post가 있음)
    ※ get
    - URL 창을 통하여 데이터를 보내는 방식
    - data 크기에 제한이 있음
    - 전송하는 데이터가 눈으로 시각화 되서 보임

    ※ post
    - http header 안에 포함하여 보내는 방식
    - data 크기에 제한이 없음
    - data가 시각화 돼서 보이지 않음 (분석을 통하여 볼 수 있음)

name : <font> 태그의 고유 이름 지정, 여러개의 form 태그를 구분하기 위해 사용

target : action 동작시 페이지를 열것인지를 결정(_self, _blank)

autocomplete : 이전 입력내용을 출력하는 기능, 생략 하면 자동출력 (default : on)
※ 자동 저장 기능 같은것
※ form 태그에 지정하면 form에 속하는 모든 input tag에 지정됨

## <fieldset> , <legend> 태그 ##
-> 폼 요소를 그룹으로 묶는 태그, 묶은 폼 요소에 명칭을 붙이는 태그


## <input> 태그 ##
-> 사용자로부터 data를 입력받기 위한 태그가

속성
type : 입력창의 타입을 결정하는 속성(text, checkbox, radio, select 등)
        - text : 한 줄 짜르 텍스트입력 창이 생김 (입력하는 내용이 그대로 보임)
        - password : 비밀번호 입력 창 (입력 시 ●●●●으로 표시 됨)
        - hidden : 사용자에게 보이지는 않지만 값을 넣을 수 있는 창
        - button : 버튼을 생성, 자체적으로는 별도 기능은 없고
                   스크립트에서 함수를 연결하여 활용 함
        - checkbox : 체크박스 생성
        - file : 파일 입력 양식 출력
        - image : 이미지 형태를 생성
        - radio : 라디오 버튼 생성
        - submit : 입력한 데이터를 다른 페이지로 넘기는 버튼 생성
        - reset : 입력한 내용을 지우는 기능을 하는 버튼 생성

        ※ type 속성의 값은 아래에서 다루도록 하겠음


## 그외 속성들 ##

value : input 요소의 기본값 표시
name : input 마다 구별할 수 있는 명칭
min/max/step : 하용하는 범위 최소값/최대값/증감값
autocomplete : 자동완성 기능
height/width : 입력창의 높이와 너비
readonly : 읽기 전용 필드
placeholder : 사용자 입력전 표시글
maxlength : 사용자가 입력 할 수 있는 최대 글자수 제한
minlength : 사용자가 입력 할 수 있는 최소 글자수 제한 (지원 안하는 브라우저가 많음)
size : 화면에서 표시하는 글자수
